Router Configuration

How to configure a wireless router.

Specifications
Motherboard
ROG STRIX B450-F GAMING Operating system
Windows 10 Wireless router
ASUS RT-AC85P

Look at the back of your wireless router and find the default access address.
Enter this URL or IP-address into your browser.

Open a web browser. You will be
redirected to the ASUS Setup Wizard. If
not, navigate to http://router.asus.com

You will be taken to the router setup wizard.

Select Automatic Setting.

If your router is not connected to the modem, select Manual Setting, or try again after connecting.

Choose a login name and password.
This will be used to log back into the router panel.

Select Default mode.

Choose No.

Choose DHCP.

Setup your WI-FI password and SSID.
This will be used to connect to the internet.

Configuration summary.

select Complete.

The router is now configured, your connection will be dropped.
Reconnect to your router

When you're finished and re-connected to the router, you will be able to further configure your router.
